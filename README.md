# 📨 RabbitMQ to Telegram

Este projeto demonstra como enviar mensagens para o **Telegram** a partir de eventos publicados no **RabbitMQ**, utilizando um modelo de mensageria **assíncrona**.

A proposta é desacoplar a produção de eventos do envio de notificações, aumentando flexibilidade, escalabilidade e resiliência do sistema.

---

## 🎯 Objetivo

- Consumir mensagens de uma fila do RabbitMQ
- Processar eventos de forma assíncrona
- Enviar notificações para um chat ou canal do Telegram
- Servir como base para estudos, testes e integrações reais

---

## 🛠️ Tecnologias utilizadas

- RabbitMQ
- Telegram Bot API
- Linguagem: *(Python, pika)*
